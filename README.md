# BPL Dream XI

## Description
A React-based application that allows users to create their ideal BPL Dream XI team. Users can add coins to their account and purchase players, with real-time alerts displayed using React Toastify to enhance user feedback and engagement.

## Technologies Used
- **React**: For building the user interface and managing component states.
- **CSS**: For styling the application, ensuring a responsive and attractive design.
- **React Toastify**: For displaying alerts and notifications to users, providing immediate feedback on actions taken.
- **JavaScript**: For implementing logic to handle coin addition and player purchases.

## Key Features
1. **Add Coins Functionality**: Users can add coins to their account, which can be used to purchase players for their Dream XI team.
2. **Buy Players Feature**: Users can select and buy players, with checks in place to ensure they have sufficient coins for the purchase.
3. **Alerts Using React Toastify**: Notifications appear whenever a user adds coins or purchases a player, providing clear feedback about the success or failure of their actions.
